# Customer-Behavior-Analysis
# Data Analytics Project

## Overview

This project demonstrates a complete **data analytics workflow** starting from data loading to final visualization and reporting. The objective is to analyze the dataset, extract meaningful insights, and present the findings using **Power BI dashboards and reports**.

The project includes **data cleaning, exploratory data analysis (EDA), SQL querying, visualization, and reporting** to support data-driven decision making.

---

## Dataset

The dataset used in this project contains structured data used for analysis. It includes multiple columns such as customer information, transactions, and other relevant attributes depending on the dataset source.

Key dataset features:

* Multiple records and attributes for analysis
* Contains missing values and inconsistencies requiring cleaning
* Suitable for performing SQL queries and visualization

---

## Tools & Technologies

The following tools and technologies were used in this project:

* **Python** – Data loading, cleaning, and analysis
* **Pandas & NumPy** – Data manipulation and preprocessing
* **Jupyter Notebook** – Writing and executing Python code
* **PostgreSQL / MySQL Server** – Running SQL queries for analysis
* **Power BI** – Creating interactive dashboards and visualizations
* **Gamma** – Creating presentation slides
* **GitHub** – Version control and project sharing

---

## Project Steps

### 1. Data Loading

* Imported the dataset using Python.
* Used **Pandas** to read and explore the data.

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Corrected data types
* Prepared the dataset for analysis

### 3. Exploratory Data Analysis (EDA)

* Analyzed data distribution
* Identified patterns and trends
* Generated summary statistics
* Created basic visualizations

### 4. SQL Analysis

* Loaded the cleaned dataset into **PostgreSQL/MySQL Server**
* Wrote SQL queries to extract insights
* Used aggregations, joins, and filtering to analyze the data

### 5. Dashboard Creation

* Imported processed data into **Power BI**
* Built interactive dashboards and visualizations
* Created charts, KPIs, and filters for better insights

### 6. Reporting

* Documented findings in a **data analysis report**
* Highlighted important insights and business conclusions

### 7. Presentation

* Created a **PowerPoint presentation using Gamma**
* Summarized key insights and dashboard visuals

---

## Power BI Dashboard

The Power BI dashboard provides interactive insights including:

* Key performance indicators (KPIs)
* Trend analysis
* Category comparisons
* Filters for detailed exploration

The dashboard helps users quickly understand the data and make informed decisions.

## Results & Insights

Some key insights from the analysis include:

* Identification of major trends and patterns in the dataset
* Important metrics affecting performance
* Data-driven insights useful for decision making

These insights are visualized through the **Power BI dashboard and final report**.

## How to Run the Project

1. Clone this repository
2. Install required Python libraries
3. Open the Jupyter Notebook file
4. Run the data loading and cleaning scripts
5. Execute SQL queries in PostgreSQL/MySQL
6. Open the Power BI file to explore the dashboard
7. Review the final report and presentation
